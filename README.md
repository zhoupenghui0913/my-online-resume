# Introduction

是一个优雅简洁的在线个人简历（仿写），支持全屏滚动，使用gulp完成自动化构建，改进工作流，是一个不错的个人信息展示页面。

访问地址：[周朋辉的在线简历](https://zhoupenghui0913.github.io/)

原代码在`src`文件中，生成内容可在`dist`文件夹查看。
